# LUX - Puzzle Game

A minimalistic black and white "Lights Out" puzzle game built as a mobile-first web app.

## Overview

Players tap tiles on a 5x5 grid. Each tap toggles the selected tile and its four neighbors. The goal is to turn all tiles off (black). The game features 30 levels of increasing difficulty with 1-3 star ratings per level. Progress is saved locally.

## Architecture

- **Frontend**: React + Vite, styled with Tailwind CSS (monochrome theme), state managed with Zustand
- **Backend**: Express API with PostgreSQL (Drizzle ORM) — backend is minimal, game logic is client-side
- **Mobile-first**: Full-screen layout, touch-optimized controls, bottom navigation, safe-area insets
- **Persistence**: Level progress (stars per level) stored in localStorage

## Key Files

### Shared
- `shared/schema.ts` — Drizzle table definition for `games`
- `shared/routes.ts` — API contract with Zod schemas

### Frontend
- `client/src/store/game-store.ts` — Zustand store: levels system, grid generation, toggle logic, win detection, progress persistence
- `client/src/components/Tutorial.tsx` — Interactive 5-step tutorial overlay
- `client/src/components/LevelSelect.tsx` — Level selection grid (30 levels, locked/unlocked, stars display)

- `client/src/components/GameCell.tsx` — Individual cell with animation

### Backend
- `server/db.ts` — PostgreSQL connection via Drizzle
- `server/storage.ts` — DatabaseStorage
- `server/routes.ts` — API routes

## Game Mechanics
- 30 levels, difficulty increases by adding more scramble moves per level
- Star thresholds scale with difficulty (each level has unique 3-star and 2-star move targets)
- Progress unlocks: complete a level to unlock the next one
- All puzzles are guaranteed solvable (generated by reversing moves from a solved state)
